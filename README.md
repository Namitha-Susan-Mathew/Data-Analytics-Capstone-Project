# Data-Analytics-Capstone-Project
A data analytics capstone project analyzing student exam performance using Python, focusing on key academic and lifestyle factors through data cleaning, feature engineering, and exploratory data analysis.

This repository presents a comprehensive Data Analytics Capstone Project focused on analyzing student exam performance using Python. The project explores how academic, lifestyle, and socio-economic factors such as study hours, attendance, motivation level, sleep patterns, and parental background influence student outcomes.

The analysis follows a structured approach, including data cleaning, preprocessing, feature engineering, and exploratory data analysis (EDA). Various visualization techniques and statistical methods are used to identify patterns, relationships, and key factors affecting exam scores.

The project aims to derive meaningful insights that can help understand student performance trends and support data-driven decision-making in the education domain.
